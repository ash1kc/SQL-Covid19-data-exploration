# SQL-Covid19-data-exploration
![covid](https://user-images.githubusercontent.com/118667586/206915346-c5c0456f-1062-4966-95e5-545b20848d0b.jpg)

I worked on a COVID-19 Data Exploration project using SQL. The objective was to analyze COVID cases, deaths, and vaccination trends across different countries and continents. I used SQL queries involving joins, aggregate functions, GROUP BY, views, and data type conversions to calculate metrics such as infection rates, death percentages, and vaccination progress. This project helped me improve my data analysis and SQL querying skills.

## Tool used to perform analysis- Microsoft SQL Server Management Studio

## SQL functions used-
1.  aggregation functions
2. sum, avg, min, max and count
3. order by, group by
4. logical functions AND and OR
5. subqueries
6. joins

* India had the highest death rate, 3.36, in June 2020.
